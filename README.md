# PSP_Program

## How to run locally
### Requirements
- NodeJS installed
- Terminal

### Installation
Run the following script in the Terminal:
```npm install```

### Run the server
Run the following script in the Terminal:
```npm start```

### Run the tests
Run the following script in the Terminal:
```npm test```

## Postman collection
You can find a Postman collection inside of ```postman``` folder with examples of HTTP requests to execute in the server.
